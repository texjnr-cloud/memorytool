# memorytool

Help me remember names and faces over time

## Setup

1. Install dependencies: `npm install`
2. Create `.env` file with your Claude API key
3. Run: `npm start`

## Features

- Face recognition and storage
- Spaced repetition learning (SM2 algorithm)
- AI-generated mnemonics for names
- Progress tracking

## Tech Stack

- React Native / Expo
- Claude API
- SuperMemo 2 algorithm
